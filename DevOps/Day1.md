# 📘 1. Introduction to Software Industry

The **Software Industry** focuses on designing, developing, testing, and maintaining software applications.

## 🔹 Key Domains
- **Web Development** – Websites & web apps  
- **Mobile Development** – Android/iOS apps  
- **Enterprise Software** – ERP, CRM systems  
- **Cloud Computing** – AWS, Azure, GCP services  
- **AI/ML** – Automation, prediction systems  

## 🔹 Key Roles
- Software Developer  
- Tester / QA Engineer  
- DevOps Engineer  
- Database Administrator (DBA)  
- UI/UX Designer  
- Project Manager

# 📘 2. SDLC (Software Development Life Cycle)

SDLC defines the **process of building software step-by-step**.

## 🔹 Phases of SDLC
1. Requirement Gathering  
2. Design  
3. Development  
4. Testing  
5. Deployment  
6. Maintenance  

---

## 🔸 Waterfall Model

A **linear and sequential model**

### Flow:
### ✔️ Advantages
- Simple and easy to understand  
- Well-defined stages  

### ❌ Disadvantages
- No flexibility  
- Late testing  
- Changes are difficult  

---

## 🔸 Agile Model

An **iterative and flexible approach**

### Flow:
- Work is divided into **Sprints (2–4 weeks)**  
- Continuous development + testing  

### ✔️ Advantages
- Flexible to changes  
- Faster delivery  
- Continuous feedback  

### ❌ Disadvantages
- Requires skilled teams  
- Less documentation

# 📘 4. Introduction to API

**API = Application Programming Interface**

It allows **two applications to communicate with each other**.

## 🔹 Types of APIs

### 1. REST API
- Most commonly used  
- Uses HTTP methods (GET, POST, PUT, DELETE)  

### 2. SOAP API
- Uses XML  
- More secure but complex  

### 3. GraphQL API
- Fetch only required data  
- Flexible  

## 🔹 API Use Cases
- Login system (frontend → backend)  
- Payment gateways  
- Fetching data (weather, maps, etc.)  
- Microservices communication

# 📘 5. Frontend vs Backend

## 🔸 Frontend (Client Side)

What users **see and interact with**

### Technologies:
- HTML  
- CSS  
- JavaScript  
- React  
- Angular  

### Example:
- Buttons  
- Forms  
- UI design  

---

## 🔸 Backend (Server Side)

Handles **logic, database, and processing**

### Technologies:
- Java  
- Python  
- Node.js  
- Databases (MySQL, MongoDB)  

### Example:
- Login authentication  
- Data processing  

---

## 🔹 Difference Table

| Feature           | Frontend              | Backend                 |
|------------------|----------------------|-------------------------|
| User Interaction | Yes                  | No                      |
| Logic Handling   | No                   | Yes                     |
| Languages        | HTML, CSS, JS        | Java, Python, Node.js   |
| Runs On          | Browser              | Server                  |
